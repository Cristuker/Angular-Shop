# learn-angular

Projeto criado para aprender e entender o funcionamento do Angular.

### Run backend project
```bash
# Enter on backend folder
$ cd backend

# Run
$ npm start
```

### Run Angular project
```bash
# Enter on frontend folder
$ cd frontend

# Run
$ ng serve --open
```

Angular.json tem as configurações do projeto
